## 3. ページ構成・コンポーネント設計

```
src/
├── app/
│   ├── layout.tsx          # ルートレイアウト（SEO・フォント設定）
│   ├── page.tsx            # メインページ（全セクションの組み合わせ）
│   └── globals.css         # グローバルスタイル（カラートークン・アニメーション）
└── components/
    ├── Header.tsx          # ヘッダー（ナビ・ハンバーガーメニュー）
    ├── Hero.tsx            # ヒーローセクション
    ├── Countdown.tsx       # カウントダウンタイマー＋祝・開催演出
    ├── EventOverview.tsx   # イベント概要（6枚カードグリッド）
    ├── Schedule.tsx        # 当日スケジュール（Coming Soon）
    ├── Access.tsx          # アクセス・Google Maps
    ├── ContactForm.tsx     # お問い合わせフォーム（Formspree連携）
    └── Footer.tsx          # フッター
```

### 3.1 ヘッダー (Header)
- サイトロゴ / タイトル表示
- ナビゲーション：TOP / 概要 / スケジュール / アクセス / お問い合わせ
- モバイル：ハンバーガーメニュー（Framer Motionアニメーション）
- スクロール連動背景切り替え

### 3.2 ヒーローセクション (Hero)
- ダークグラデーション背景 + グローオーブアニメーション
- キャッチコピー：「あの頃の絆を、もう一度。」
- バッジ：「🎤 特別ゲスト参戦決定！？」「🎁 豪華景品抽選会あり」
- カウントダウンタイマー（2027年1月10日 18:00まで）
- CTAボタン → お問い合わせセクションへスクロール

### 3.3 カウントダウン (Countdown)
- フリップアニメーション付きタイマー（DAYS / HRS / MIN / SEC）
- 開催時刻到達時：クラッカー紙吹雪＋「祝・開催」オーバーレイ表示

### 3.4 イベント概要 (EventOverview)
- 6枚カードグリッド（日時 / 会場 / 会費 / 対象 / 服装 / イベント内容）
- 先生注釈：「※恩師の先生方からは徴収いたしません」

### 3.5 スケジュール (Schedule)
- Coming Soonプレースホルダー（詳細決定後に更新）

### 3.6 アクセス・地図 (Access)
- Google Maps 埋め込み（プラトンホテル四日市）
- 電車（近鉄四日市駅より徒歩約3分）・車でのアクセス案内

### 3.7 お問い合わせフォーム (ContactForm)
- 入力項目：お名前（必須）/ 返信用メールアドレス（必須）/ お問い合わせ内容（必須）
- 送信先：Formspree 経由で `koshi1250@icloud.com` に自動転送
- 件名自動設定：「【保々中同窓会】{名前}さんからのお問い合わせ」
- 送信完了画面・エラーハンドリング実装済み

### 3.8 フッター (Footer)
- ロゴ / イベント名
- お問い合わせメール：`koshi1250@icloud.com`
- コピーライト：© 2027 保々中学校 同窓会実行委員会

## 4. デザイン要件

### 4.1 カラーパレット
- プライマリ：ネイビー〜ディープブルー（`#0f172a` 〜 `#1e3a5f`）
- アクセント：ゴールド / アンバー（`#f59e0b` 〜 `#d4a574`）
- 背景：ダークグラデーション（`#050510` → `#0a0a12`）
- テキスト：ホワイト / ライトグレー

### 4.2 タイポグラフィ
- 日本語：Noto Sans JP（Google Fonts）
- 英数字：Inter（Google Fonts）

### 4.3 アニメーション
- Framer Motion によるフェードイン・スライドイン（スクロール連動）
- カウントダウンフリップエフェクト
- ホバーエフェクト（ボタン・カード）
- 紙吹雪パーティクル（開催到達時）
